CHIMERA

A desktop app that makes scientific data analysis fast and painless — load data, fit a model, and export a publication-quality figure in a few clicks.

Project site & download ↗ Source on GitHub ↗

The motivation

Every physics student ends up doing the same tedious loop: paste data into some tool, wrangle a fit, fight with error bars, and re-export the plot every time a number changes. We surveyed peers (49 responses) and confirmed the pain was widespread — then built CHIMERA to remove it: a single, friendly GUI covering the whole path from raw data to a clean, labelled figure.

What it does

  • Curve fitting done right — fits use orthogonal distance regression (SciPy ODR), so uncertainties in both axes are handled properly, not just y.
  • Publication-quality plots — Matplotlib figures with full control over styling, labels, and export.
  • Frictionless data entry — paste tabular data straight in (pandas under the hood), pick columns, go.
  • Built to be shipped — packaged with PyInstaller into standalone apps for Windows and Linux, with an in-app version check for updates.

Designed as a real product, not just a script — the team ran a public-interest survey, iterated on name and branding, and released versioned, installable builds with a dedicated project website.

My role

A three-person team (Leonardo Alves, Tomás Feith, Paulo Machado). I worked across the stack — the Tkinter interface, the fitting and plotting engine, and the build/packaging and release pipeline.
